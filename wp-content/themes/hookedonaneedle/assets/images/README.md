# Theme Images

This folder contains theme images and placeholders.

## Image Requirements

- **hero-bg.jpg** - Hero section background (1920x1080 recommended)
- **hero-default.jpg** - Default hero featured image (1200x1600 recommended)
- **placeholder.jpg** - Generic placeholder for missing images (800x1000)
- **featured-1.jpg** - Featured creation 1 (800x1000)
- **featured-2.jpg** - Featured creation 2 (800x1000)
- **featured-3.jpg** - Featured creation 3 (800x1000)

## Notes

Default images are used as fallbacks when ACF fields are empty.
Replace with actual images for production use.
